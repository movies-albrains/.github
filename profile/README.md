# Resume
Web application allowing user to borrow a movie.

# Main requirements
- Authenticate a user 
- Borrowing a movie

# Tech stack
- Backend: Java, Kotlin
- Frontend: React
- Database: PostgreSQL
- Auth: Keycloak, OAuth2

# Modeling practices
- Architecture document
- Domain driven design
- Modeling tools: UML, Archimate
- Document as code: Context Mapper
